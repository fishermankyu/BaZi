# 神煞与空亡 (Special Stars & Void) — Supplementary, Not Structural

Both 神煞 (special stars) and 空亡 (the void) sit **outside** the 十神/五行生克 logic that everything else in this skill is built on. They're not derived from how an element relates to the Day Master — they're fixed lookups: a specific stem or branch combination directly maps to a named marker, each with its own traditional meaning. Treat this file as supplementary color, not part of the core structural analysis (格局/扶抑/従格/调候/通关) — don't let it override or compete with those conclusions.

**When to use this file**: only when the person specifically asks about a named 神煞 (桃花/驿马/华盖 etc.), asks "我八字里有什么" in an open-ended way, or asks about 空亡. Don't proactively surface this in an ordinary structural reading — it's opt-in, same as `dayunliunian.md`.

**Standing caveat for this whole file**: 神煞 easily invites "ooh, I have that!" pattern-matching without connecting it back to whether it's landing on something the chart favors or resents. Per source material's own warning: *先从十神制化分析,神煞可以作为旁支参考,不可舍本逐末*. Always frame conclusions relative to whether the position/stem involved is a 喜用神 or 忌神 per `zonghecaijue.md` — the same marker reads differently depending on that.

## 1. Common 神煞 — lookup rules

This covers the widely-used core set, not the full traditional list (which runs into the hundreds across schools) — flag that more exist if asked for something not listed here.

|---|---|---|

| **文昌贵人** | By Day stem (or year stem): 甲→巳, 乙→午, 丙→申, 丁→酉, 戊→申, 己→酉, 庚→亥, 辛→子, 壬→寅, 癸→卯. | Intelligence, academic/literary aptitude, quick recovery from setbacks. |

(This is the 沐浴 position of that branch's 三合 bureau.) | Charisma, romantic/social magnetism. ## 2. 空亡 (The Void)

### Concept

The 60-ganzhi cycle (六十甲子) splits into six groups of ten (旬), each starting on a 甲-stem day. Since there are 10 stems but 12 branches, each group of ten pairings uses only 10 of the 12 branches — the two branches left unpaired in that group are that group's **空亡 (void)**.

```
甲子旬 → 戌亥空
甲戌旬 → 申酉空
甲申旬 → 午未空
甲午旬 → 辰巳空
甲辰旬 → 寅卯空
甲寅旬 → 子丑空
```

### How to check

Modern mainstream convention: **use the Day pillar** (older texts also cross-check the Year pillar, but Day-pillar-based checking is what most practitioners default to today — mention this if asked, don't present it as the only method). Find which of the six 旬 groups the Day pillar belongs to, read off that group's two void branches, then check whether either appears among the **other** three pillars' branches (year/month/hour — not the day branch itself, which can't be void relative to its own pillar).

### What it means

A branch in 空亡 is read as weakened, "not fully substantial" for the domain that pillar represents — not erased, but under-delivering relative to what it would otherwise suggest.

| Position | Domain affected |
|---|---|
| 年支空亡 | Early life / ancestors / family backing — less inherited support, more self-made |
| 月支空亡 | Siblings / peer relationships; some sources also read this as diffuse restlessness in early-to-mid life |
| 日支空亡 | Spouse / marriage — described as a thinner or more distant marital connection |
| 时支空亡 | Children / late life — less support from or connection to children in later years |

The Ten God occupying that void branch also colors the specific reading (e.g. 印 in a void position leaning toward "less closeness with mother / interrupted formal education," 财 in a void position leaning toward "money doesn't stay" — cross-reference `ten_gods.md`'s categories for which life domain each Ten God maps to, rather than treating 空亡 as a separate meaning system).

### Modifiers — when 空亡 is strengthened or lifted

- **空亡逢冲**: source material states a clash on the void branch can "冲空亡之字...空亡去" — resolving/lifting the void. There's a documented **exception and counter-view** worth flagging: some sources specifically note that if the void branch itself is already very weak (休囚 per its seasonal state), a clash on it can instead precipitate misfortune rather than resolve the void — i.e. clashing something already faint can break it rather than fix it. This is a genuine point of nuance/disagreement between sources, not a settled rule — say so if asked rather than picking one side confidently.
- **空亡逢合** (六合/三合/三會): generally **not** treated as void in this case — a combination is read as substantial enough to override the void designation.
- **大运/流年填实**: if a later 大运 or 流年 brings the exact void branch back in, that's called "填实" (filling it in) — the domain that was under-delivering can become active again for that period. Note: 流年 itself doesn't have its own void (空亡 is a property of the original chart's Day pillar, and by extension the 大运's own day-pillar-equivalent framing isn't typically computed this way either) — void only applies to reading the **original chart**, with 大运/流年 acting as things that can "fill" it, not carry their own void.
- **Direction relative to 喜用神/忌神**: same "which side does this hit" logic as everywhere else in this skill — a 喜用神 landing in 空亡 is weakened (unwelcome), while a 忌神 landing in 空亡 is suppressed (welcome). Never call 空亡 uniformly bad without checking which side it's hitting.

## Integration note

Both 神煞 and 空亡 are read **after** the core structural analysis (格局→扶抑→従格→调候→通关→synthesis), as supplementary color when specifically asked. Don't let a notable 神煞 (e.g. 魁罡, 天乙贵人) reframe or override what `zonghecaijue.md` already concluded about the chart's structure — these add texture to an already-established reading, they don't compete with it.

## Worked example (using the chart already run earlier: 己卯 戊辰 壬子 己酉)

**神煞found**: 天乙贵人 (壬 sees 卯 in the year branch; 己 sees 子 in the day branch — both landing on core pillar positions, not edge cases), 文昌贵人 (己 sees 酉 in the hour branch), 桃花 (双向: 卯→子 and 子→酉, forming a closed loop across three of the four branches), 华盖 (子 sees 辰 in the month branch), 羊刃 (日支坐禄旺, day branch 子 is 壬's 帝旺 position). 驿马/将星/魁罡 absent. 天德/月德 technically land on 壬 itself (the Day Master) — an edge case some schools don't count as a "real" hit since it's self-referential; noted but not confidently claimed.

**Notable pattern**: 年支卯 alone carries 天乙贵人 + one leg of 桃花, while *also* being the branch identified in `xingchonghehui.md`'s earlier analysis as the most heavily-clashed, most rootless branch in the whole chart (final multiplier 0.4, hit by 沖/刑/害 simultaneously). A branch can carry a "prestigious" marker and be structurally the weakest point in the chart at the same time — these are two different information layers, not a contradiction to resolve, and both should be surfaced rather than only reporting the flattering one.

**空亡 check**: Day pillar 壬子 falls in 甲辰旬 → void branches are 寅, 卯. Checking the other three branches (卯, 辰, 酉): **年支卯 is void**.

**Interacting modifiers, genuinely in tension**: 卯 carries 天乙贵人 (per the "空亡逢贵神减力" convention, this softens the void's negative reading), but 卯 is also clashed (卯酉冲) and was already established as structurally weak — and per the documented disagreement in §2's modifiers section, a clash on an already-weak void branch may aggravate rather than resolve the void. Both readings are live here; the honest answer is that they pull in different directions rather than one cleanly winning, and that's worth saying plainly rather than picking a side to sound more decisive than the framework actually supports.
